# Simple Data Web Application using Python and Streamlit

Mini project ini dikembangkan sebagai bagian dari praktik pembuatan aplikasi web data interaktif menggunakan Python dan Streamlit. Proyek ini mengintegrasikan konsep dasar pemrograman, antarmuka pengguna (UI), dan sistem manajemen basis data ke dalam satu aplikasi *full-stack* sederhana.

## 📝 Deskripsi Proyek
Aplikasi ini berfokus pada pengolahan dan visualisasi data secara interaktif. Dibangun dengan pendekatan *Object-Oriented Programming* (OOP) menggunakan Python murni pada sisi *backend* dan Streamlit untuk merender *frontend* secara instan tanpa memerlukan HTML/CSS/JS tambahan.

Proyek ini merupakan hasil implementasi dari materi workshop **"Building a Simple Data Web Application using Python and Streamlit"** di Cakrawala University oleh Faridho (Lead Software Engineer at Lion Group).

## ✨ Fitur Utama (Learning Points Implemented)
* **Python OOP & Data Structures:** Logika bisnis dan pengolahan data menstrukturkan fundamental Python yang solid.
* **Interactive UI with Streamlit:** Antarmuka responsif dan interaktif yang dibangun secara instan menggunakan Streamlit (*zero front-end hassle*).
* **Database Integration:** Aplikasi terhubung dengan sistem penyimpanan data (Database) untuk proses CRUD (Create, Read, Update, Delete) yang dinamis.
* **Full-Stack Integration:** Penggabungan sempurna antara kode *backend*, *database*, dan *UI* ke dalam satu ekosistem aplikasi dunia nyata.
